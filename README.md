## 프로젝트 개요

SNU 반려동물 검진센터 시스템은 반려동물의 건강 데이터를 체계적으로 관리하고 분석하기 위한 웹 애플리케이션입니다.

### 주요 목표

- 📊 **데이터 중심 관리**: 검사 결과, 방문 기록, 문진 응답 등을 체계적으로 저장 및 관리
- 🔍 **고급 필터링**: 품종, 나이, 검사 결과 등 다양한 조건으로 환자 필터링
- 📈 **시각화**: 건강 데이터를 차트와 그래프로 직관적으로 표현
- 💊 **약물 관리**: 처방 약물 및 투여 기록 추적

## 주요 기능

### 환자 관리
- 환자(반려동물) 등록 및 정보 관리
- 동물번호, 품종, 나이, 성별, 중성화 여부 등 상세 정보 저장
- 환자별 통계 및 건강 이력 조회

### 검사 결과 관리
- CBC, 간기능, 신장기능, 전해질 등 다양한 검사 항목 지원
- 정량적/정성적 검사 결과 입력
- 정상 범위 대비 상태 표시 (High/Normal/Low)
- 검사 결과 시각화 (차트, 레이더 그래프)

### 방문 기록
- 방문 일자, 유형, 주증상, 진단, 처치 내역 기록
- 방문별 검사 결과 및 약물 처방 연결

### 문진 관리
- 카테고리별 문진 템플릿 (피부, 소화기, 호흡기, 신경, 비뇨기, 근골격)
- 문진 응답 기록 및 이력 조회

### 필터 및 군집 분석
- 사용자 정의 필터 생성 및 저장
- 품종, 나이, 검사 결과 기반 군집 분석
- 저장된 필터 재사용

## 기술 스택

### Frontend
- **React 18** - UI 라이브러리
- **TypeScript** - 타입 안전성
- **Vite** - 빌드 도구
- **Wouter** - 경량 라우팅
- **TanStack Query (React Query)** - 서버 상태 관리
- **Zustand** - 클라이언트 상태 관리
- **Tailwind CSS** - 스타일링
- **Radix UI** - 접근성 높은 UI 컴포넌트
- **Recharts** - 데이터 시각화

## 시작하기

### 사전 요구사항

- **Node.js** 20.x 이상
- **npm** 또는 **yarn**

### 설치

1. **저장소 클론**

```bash
git clone <repository-url>
cd SNUClinicSystem
```

2. **의존성 설치**

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

서버가 시작되면 브라우저에서 http://localhost:5173 으로 접속합니다.

### 프로덕션 빌드

```bash
npm run build
npm run preview
```

## 개발 가이드

### 타입 체크

```bash
npm run check
```

### Mock Data 사용

현재 프로젝트는 백엔드 없이 Mock Data를 사용하여 동작합니다. `client/src/lib/api.ts` 파일에서 Mock Data를 수정하거나 추가할 수 있습니다.
